# Medi Explain AI 🏥🤖

## 📌 Project Title
Medi Explain AI – An Intelligent Medical Report Analysis and Explanation System

## 🎓 Course
MCA Industrial Training (4th Semester)

## 📖 Introduction
Medi Explain AI is a web-based application designed to help users understand their medical reports easily. The system uses OCR and Artificial Intelligence to extract and analyze medical data and provides explanations in simple language.

## 🎯 Objective
- To simplify medical reports for normal users
- To extract text from PDF/images using OCR
- To analyze health parameters using AI
- To provide easy explanations

## ⚙️ Technologies Used
- Python
- Flask
- MySQL
- Tesseract OCR
- HTML, CSS, JavaScript

## 🔍 Features
- Upload medical reports (PDF, JPG, PNG)
- OCR text extraction
- Data cleaning & processing
- AI-based explanation generation
- Download analyzed report
- User-friendly interface

## 🧠 Working
1. User uploads report
2. OCR extracts text
3. System processes data
4. AI analyzes parameters
5. Output shown in simple language

## 🔐 Security
- User authentication
- Encrypted passwords
- Secure file handling

## 📊 Project Modules
- User Interface Module
- OCR Module
- Text Processing Module
- AI Explanation Module
- Database Module
